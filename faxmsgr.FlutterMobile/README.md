# faxmsgr.FlutterMobile — Мобильное приложение FAX Messenger

Мобильный клиент мессенджера FAX. Стек: **Flutter**, **Dart**.

## Требования

- Flutter SDK 3.x+
- Android SDK / Xcode (для iOS)

## Разработка

```bash
flutter pub get
flutter run
```

## Сборка

```bash
# Android
flutter build apk

# iOS
flutter build ios
```

## Конфигурация

URL сервера задаётся в конфигурационном файле приложения (будет добавлено).
